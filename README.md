# Arch 

## Features
- OS : Arch Linux
- Window Manager : Bspwm
- Shell : fish
- Bar: Polybar
- Editor : Neovim
- Compositor : picom
- Terminal : Alacritty
- Window Switcher : rofi
- Audio Player : cmus
